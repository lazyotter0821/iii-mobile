# 个人所得税计算器

一个功能完整的个人所得税计算器，支持电脑端和移动端访问，适用于计算综合所得和年终一次性奖金的个人所得税。

## 功能特点

### 核心功能
- 全年工资薪金和年终一次性奖金输入
- 五险一金计算（含计算缴纳金额和实际缴纳金额）
- 专项附加扣除（含开始享受月份和享受标准）
- 年终奖单独计税和并入综合所得计算
- 个人所得税累计预扣计算
- 图表展示（各月个人所得税和累计预扣趋势）
- 对比结论（推荐最优计税方式）
- 税率表（累计预扣税率表和年终奖税率表）

### 技术特点
- **电脑版**：适合桌面端大屏幕显示，左右布局，内容丰富
- **手机版**：响应式设计，垂直布局，触摸友好
- **纯前端实现**：使用HTML、CSS和JavaScript，无需后端服务
- **数据可视化**：使用Chart.js绘制图表
- **实时计算**：修改参数后自动重新计算

## 主要参数说明

### 收入信息
- 全年工资薪金总额：输入全年的税前工资总额
- 年终一次性奖金：输入年终奖金额
- 年终奖发放月份：选择年终奖发放的月份

### 五险一金
- 缴纳比例：各险种的个人缴纳比例
- 缴纳基数：各险种的缴纳基数
- 本年计算缴纳金额：根据比例和基数自动计算的全年金额
- 本年实际缴纳金额：实际缴纳的全年金额

### 专项附加扣除
- 开始享受月份：选择开始享受专项附加扣除的月份
- 享受标准（元/月）：每月的扣除标准
- 金额（元/年）：根据开始月份和标准自动计算的全年金额

### 其他
- 免税收入（元/月）：每月的免税收入
- 本年实际已预交个人所得税（元）：已经预交的个人所得税

## 计算逻辑

### 1. 累计预扣法
按照中国个人所得税法规定的累计预扣法计算工资薪金的个人所得税。

### 2. 年终奖计算
- **单独计税**：将年终奖除以12个月，确定适用税率和速算扣除数，单独计算个人所得税
- **并入综合所得**：将年终奖并入全年工资薪金总额，合并计算个人所得税

### 3. 对比分析
自动对比两种计税方式的税额，推荐更优的计税方式。

## 注意事项

1. 本计算器仅适用于综合所得（工资、薪金所得 / 劳务报酬所得 / 稿酬所得 / 特许权使用费所得）和年终一次性奖金
2. 适用税收法规截止2026年3月31日
3. 计算结果仅供参考，实际纳税以税务部门核算为准
4. 如有疑问，请咨询专业税务顾问
